# Cheddar-Chase
Cheddar Chase é um pequeno jogo 2D criado no GameMaker como meu primeiro projeto para aprender a engine e explorar o processo de desenvolvimento de jogos.  No jogo, você controla um rato que precisa coletar 5 pedaços de queijo enquanto desvia de ratoeiras móveis e fixas. Ao pegar todos os queijos sem ser atingido, você avança para a próxima fase.
